方法

__construct()
-------------
构造函数

.. code-block:: php

  public function __construct (
      string $url = null,
      array $opt = [],
      bool $share = false
  )


:参数:
  +-------+------------------------+
  |名称   |说明                    |
  +=======+========================+
  |url    |指定会话链接            |
  +-------+------------------------+
  |opt    |指定选项                |
  +-------+------------------------+
  |share  |指明是否使用share       |
  +-------+------------------------+
